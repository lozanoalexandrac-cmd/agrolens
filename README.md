# AgroLens Landing Web

Landing web estática para presentar **AgroLens** como proyecto de tesis y validar el interés del mercado.

## Contenido

Esta web está diseñada para explicar de forma clara y rápida:

- el problema que busca resolver AgroLens,
- la solución propuesta,
- cómo funcionaría el flujo del dron al dashboard,
- la propuesta de valor,
- los beneficios tentativos,
- y ejemplos visuales del dashboard y reporte.

## Cómo usarlo

### Opción 1: abrir localmente
Solo abre `index.html` en tu navegador.

### Opción 2: subirlo a GitHub Pages
1. Crea un repositorio en GitHub.
2. Sube todo el contenido de esta carpeta.
3. Ve a **Settings > Pages**.
4. En **Build and deployment**, elige **Deploy from a branch**.
5. Selecciona la rama principal y la carpeta `/root`.
6. Guarda los cambios.

GitHub Pages publicará la web automáticamente.

## Personalización rápida

Puedes editar fácilmente:

- **Textos**: en `index.html`
- **Colores y estilos**: en `assets/css/styles.css`
- **Interacción del menú móvil**: en `assets/js/main.js`
- **Imágenes**: en `assets/images/`

## Notas

- La web usa únicamente HTML, CSS y JavaScript puro.
- Las imágenes incluidas son simulaciones visuales creadas para ilustrar el concepto del proyecto.
- El enfoque es informativo y comercial, pero sin sobreprometer resultados.
